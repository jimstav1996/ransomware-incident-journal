# 🛡️ Ransomware Incident Journal

## 📌 Overview
This project documents the initial incident response journal entry for a ransomware attack against a healthcare clinic.

## 📧 Scenario
A small U.S. healthcare clinic experienced a ransomware attack after multiple employees received targeted phishing emails containing a malicious attachment.

Once downloaded, the malware was executed and encrypted critical patient data and business systems.

## 🎯 Objective
- Document the incident using the 5 W's
- Capture initial observations
- Support SOC and incident response workflows
- Practice ransomware triage documentation

## 🛠️ Skills Demonstrated
- Incident documentation
- Ransomware triage
- The 5 W's framework
- SOC analyst workflow
- Security reporting
